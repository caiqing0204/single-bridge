# typeof与instanceof区别

typeof 多用于判断原始数据类型：number、string、null、undefined、boolean、symbol、function

instanceof 多用于判断引用数据类型： Regexp、Object、Array

## typeof

当数据是function时，会返回'function'

typeof null ==== 'object'

## instanceof

instanceof 只能校验某个对象是否是某个类的实例
